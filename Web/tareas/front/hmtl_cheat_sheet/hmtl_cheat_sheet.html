a- hyperlink

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web

por ejemplo: cr7highlights

p- párrafo

Sirve para definir un párrafo

por ejemplo:
Hola, hoy hubo un eclipse

También se puede alinear:

Este texto está lineado a la derecha

h1, h2, ... , h6 : encabezados

Son usados para definir los encabezados de la página.
H1 es el encabezado más importante, solo se puede usar una vez, apartir de ahí se pueden usar los demás las veces necesarias, pero no te debes saltar niveles.
Es decir, si usaste h2, va h3, no debes irte de la nada a h6. Por ejemplo:

encabezado 1

encabezado 2

encabezado 3

encabezado 4

encabezado 5
encabezado 6

div

Sirve para definir una division o sección
Usada como contendor de los elementos de HTML, se pueden después estilizar con CSS.
Por ejemplo:

cr7 es el mejor jugador de todos los tiempos

span

Es un contendor en HTML que se utiliza para aplicar estilos específicos a partes del contenido dentro de un elemento sin afectar su significado o estructura.

Por ejemplo:
En este párrafo, utilizamos span para resaltar una parte del texto.

strong

Se utiliza para enfatizar un texto importante en el documento

Este texto es normal.

¡Este texto es importante!

i

se utiliza para incluir texto en cursiva o itálica. Se recomienda su uso semántico para palabras o frases que deben ser presentadas de manera diferente del texto circundante

Por ejemplo:
Texto normal y... Texto inclinado >

u

Se usa para representar texto no articulado o mal escrito. El texto sale subrayado

Por ejemplo:
Esto está mal escritu

b

La etiqueta especifica texto en negrita sin ninguna importancia adicional.

Por ejemplo:
Este es texto normal - y este es texto en negrita.

strike

Se utiliza para mostrar texto tachado en un documento HTML. Es comúnmente utilizado para indicar que un texto ha sido eliminado o es obsoleto


Ejemplo de uso:

En este párrafo, utilizamos strike para mostrar texto tachado.

small

La etiqueta small define texto más pequeño (como derechos de autor y otros comentarios laterales).


Ejemplo de uso:

Este es un texto normal.

Este es un texto más pequeño.

em

La etiqueta em se utiliza para definir texto enfatizado. El contenido interior normalmente se muestra en cursiva.


ejemplo de uso:
tú tienes que apurarte

header

El elemento header representa un contenedor para contenido introductorio o un conjunto de enlaces de navegación.
Normalmente contiene los elementos de encabezado, logotipo o información de autoría.


Ejemplo:

Encabezado Principal

  • Inicio
  • Acerca de
  • Contacto

section

La etiqueta section define una sección en un documento.

Ejemplo:

Sección 1

Contenido de la sección 1.

Sección 2

Contenido de la sección 2.

article

La etiqueta article especifica contenido independiente y autónomo.


por ejemplo:

10 mejores champions de la historia

Contenido del artículo sobre las 10 mejores champions de la historia

Eciplse lunar

Contenido del artículo sobre el eclipse lunar.

aside

La etiqueta aside define algún contenido aparte del contenido en el que se encuentra. El contenido aparte debe estar indirectamente relacionado con el contenido.

Ejemplo:

Draft NFL

Contenido del artículo sobre Draft NFL.

La etiqueta footer define un pie de página para un documento o sección.
